About This Project

Purpose

The purpose of this analyzer is to provide users with a quick way to check if the URL may pose any security risk. It's suporse is to quickly scan a site with using the following APIs to identify if there is any phising attempts or malware attached to the site.

APIs Used

  • URLScan.io
  • PhishStats
  • anime.js
  • Chart.js

Functionality

Users enter the web URL into the scan page which is then analyzed by checking the db and scan for potential malware. It will then classify it depending on if it's dangerous or not. Results may not always be aviable if the site hasn't been added or tracked on the sites before.
